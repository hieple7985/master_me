# Thư mục /ideas

Nơi ghi chú nhanh các **IDEA (Ý tưởng)** chợt đến.

## Gợi ý tổ chức

- Đặt tên file theo: `YYYY-MM-DD-ten-y-tuong.md`
- Mỗi file nên có:
  - Tiêu đề
  - Ngữ cảnh (tại sao xuất hiện)
  - Mức độ ưu tiên / tiềm năng
  - Các bước tiếp theo nếu có

## Luồng sử dụng

- Ghi nhanh, không cần hoàn hảo
- Ý tưởng nào bắt đầu được triển khai / nghiên cứu sâu hơn thì tạo entry tương ứng trong `/rnd/`
