# game-05. Solomon 72 Demonds

## Decisions

- (B) "The Seventy-Two" = 72 sigils/contracts (nội dung/hợp đồng), không phải 72 boss AAA.

## Product Option

- Option 1: Soulslike Boss-Rush Premium (hub + chọn hợp đồng boss/hunt + craft từ sigils).

## Engine

- Unreal Engine (ưu tiên đồ họa đẹp, chất lượng cao).

## Working Title

- Seals of Solomon: The Seventy-Two

## Core Pitch

- Boss-driven game loop: hạ boss -> nhận sigil/loot -> craft/mở build -> nhận contract mới -> boss mạnh hơn.
- "72" là hệ thống nội dung (sigils/contracts) để mở rộng lâu dài và làm endgame, không bắt buộc 72 boss cinematic ngay bản v1.

## Vibe / Direction (đề xuất)

- Occult dark fantasy theo hướng "Sealkeeper" (restore/banish), tránh cảm giác "thờ quỷ".
- Visual pillars:
  - Sigils / sacred geometry (ấn triện, vòng khắc, hình học thiêng)
  - Vellum/ink + iron (giấy da, mực, kim loại, xiềng xích)
  - Prison-architecture (library-prison, đàn tế, đại sảnh phong ấn)

## Content Scope Target (2027)

- 12 major bosses (arena + phase 2 rõ ràng).
- 24-36 hunts/variants (modifier + moveset remix + điều kiện contract).
- ~30 vũ khí/stance hoặc ít hơn nhưng “chất” (tùy pipeline).
- 3-5 schools phép/skills dạng seals/sigils (mỗi school có build identity rõ).

## System Notes

- Reward structure (khung): Kill boss -> Sigil (key item) + Core (upgrade) + Oath Fragment (lore/questline).
- Hub: craft vũ khí/giáp/phép theo sigil; chọn contract/hunt; build management.
- Tránh reskin boss: mỗi boss cần 1 combat idea rõ (defense check + offense pattern + arena twist).

## Production Plan (2 years) - High Level

- M0-M3: Combat Vertical Slice
  - 1 playable character + 1 boss “đinh” + 1 arena + 3 vũ khí/stance.
- M4-M8: Production Pipeline
  - Boss framework (AI/state machine), hitbox/hurtbox rules, telegraph, audio cues.
  - Thêm 2-3 boss để chứng minh pipeline.
- M9-M14: Content Build
  - Đạt 8 boss playable end-to-end.
- M15-M20: Polish + Demo (wishlist)
  - Demo public (Steam Next Fest) + trailer/marketing beats.
- M21-M24: Content finalize + QA + Release

## Go-To-Market Notes (nháp)

- Steam-first (PC) để iterate nhanh.
- Boss-rush thường hợp Full release hơn Early Access (tránh bị chê “thiếu content”), nhưng vẫn cần demo sớm để kéo wishlist.

## Highest Win-Rate Release Strategy (2027)

- 2027: PC (Steam) Full Release.
- 2026-2027: làm demo sớm để kéo wishlist (Steam Next Fest).
- 2027-2028: patch/polish + tối ưu performance để sẵn sàng port.
- 2028: port PS + Xbox (launch marketing vòng 2).
- Nintendo: cân nhắc theo performance/memory budget, có thể ra muộn hơn như "definitive edition".
