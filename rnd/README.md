# Thư mục /rnd

Nơi lưu trữ **RND (Research & Development / Nghiên cứu & Thử nghiệm)**.

## Nội dung phù hợp

- Phân tích chi tiết một ý tưởng
- Thử nghiệm, POC, prototype
- Tài liệu nghiên cứu, tổng hợp kiến thức

## Gợi ý tổ chức

- Có thể nhóm theo chủ đề hoặc project
- Đặt tên file theo: `YYYY-MM-DD-chu-de-rnd.md`
- Mỗi file nên có:
  - Mục tiêu
  - Giả thuyết / câu hỏi nghiên cứu
  - Ghi chú quá trình thử nghiệm
  - Kết luận và bước tiếp theo

## Luồng sử dụng

- Bắt nguồn từ một IDEA
- Sau khi ổn định / hoàn thành có thể liên kết sang tài liệu chính hoặc repo khác
